PropertyPal

A desktop app for real estate professionals to track, filter, and prioritise clients in one place..

Overview

PropertyPal is a desktop application developed as part of the CS2103 Software Engineering course at the National University of Singapore. It helps real estate agents manage client information and property preferences efficiently through a fast, command-driven interface. PropertyPal enables users to add, search, filter, and organise client records such as contact details, property types, prices, and intentions, making client management quicker and more structured.

View GitHub repo →

Features

  • Add, edit, and delete client contact records
  • Fast command-based search across multiple fields
  • Filter clients by intention, property type, and price
  • Persistent local storage with automatic saving

Technologies

  • Java 17
  • JavaFX (desktop GUI)
